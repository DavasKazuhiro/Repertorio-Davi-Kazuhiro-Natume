Controle de Estoque para ONG

Site de controle de estoque desenvolvido em HTML e CSS para auxiliar a ONG Crazy Cat Gang responsável por resgatar, cuidar e enviar os gatos para um novo lar. O site tem como objetivo tornar mais fácil o monitoramento do estoque de insumos utilizados pelos gatos e a logística de envio dos mesmos para clínicas e LT's
